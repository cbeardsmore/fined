So how does Fined work?

Click the button below to add FinedBot to your Slack Workspace.

  • Usage

    Use /fines help to explain usage and provide a list of common commands.

  • Display Fines

    To display all the fines that are unpaid in the current channel use /fines.

  • Fine a teammate

    /fines @Tim $5 for being late to standup... again

  • Paying fines

    First display the channels fines using /fines. Users can mark their fines as paid so it is up to you to keep people honest.
